# L0-Gateway

## About

Distributed API for distributed finance.

## Getting Started

```
git clone https://github.com/ProjectL0/L0-Gateway
```

### Features 
- `default` : all Features
- `eth` : enables ethereum service

### Default Ports
- `8810` : gateway-local
- `8811` : gateway-provider
- `8820` : ethereum api
- `8821` : ethereum p2p
